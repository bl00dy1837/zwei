# Zwei

Game engine for 2d action RPG style games.

- [x] Layers, Sprites, Animations
- [x] Import Maps and Tilesets from [Tiled](https://www.mapeditor.org/)
- [x] Items, Weapons and Inventory
- [x] Combat
- [x] Menu
- [ ] Save & Load 
- [ ] Scripting
- [ ] Building on Windows and Mac OS
